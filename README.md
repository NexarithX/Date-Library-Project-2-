# Date & String Utility Library (C++)

A custom-built C++ utility library that provides robust Date manipulation and advanced String operations.

This project was developed to demonstrate strong understanding of:
- Object-Oriented Programming (OOP)
- Encapsulation & Clean Class Design
- Constructor Validation
- Operator Overloading
- Static Member Functions
- Robust Error Handling

---

## 📅 Date Class Features

- Date validation (prevents invalid object creation)
- Leap year detection
- Date comparison operators (==, !=, <, >, <=, >=)
- Increase/decrease date (days, months, years)
- Calculate difference between two dates
- End of week/month calculations
- Period handling support

---

## 🔤 String Utility Features

- Custom string validation
- Parsing & formatting
- Flexible input handling
- Utility helper functions

---

## 🛠 Technologies Used

- C++
- Object-Oriented Design
- Modern C++ Features

## 🎯 Purpose of the Project

This project was built from scratch as part of improving core C++ programming skills and understanding how standard libraries work internally.

Instead of relying on built-in date or string libraries, the goal was to implement everything manually for learning and practice.

---

## 🚀 Example Usage

```cpp
clsDate date1(10, 5, 2024);
clsDate date2(15, 5, 2024);

if (date2 > date1)
{
    cout << "Date2 is later than Date1\n";
}
```
